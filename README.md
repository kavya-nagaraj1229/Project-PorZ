Project Overview:
-----------------
Nutrifit is a modern and user-friendly landing page design for a weight loss and calorie tracking application. The design focuses on simplicity, motivation, and ease of use, helping users track food, set goals, and live healthier. This UI is inspired by a clean wellness theme with soft colors, illustrations, and clear call-to-action buttons.
